# lo-bidding-api
This is an api which exposes methods to create auction, make a bid and determine/get the winner. All bids should be done using cents.

##TECH STACK:
    Java 17
    Gradle 7.3
    Lombok 1.18.22
    JUnit 5.7.0

## HOW TO BUILD
    gradle clean build

## HOW TO RUN TESTS
    ./gradlew test
